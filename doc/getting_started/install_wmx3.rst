Install WMX3
====================

WMX3 is MOVENSYS's software-defined motion control stack.
It connects a PC to servo drives via Lan cable and provides deterministic cycle loop.


.. rubric:: Prerequisites

1. Ubuntu 22.04/24.04 LTS
#. ``sudo`` privileges
